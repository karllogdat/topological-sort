About the Project

The PUP Curriculum Navigator is designed to help self-learning students navigate the complex interdependencies of Computer Science topics. Using topological sorting algorithms, we create a structured and logical curriculum that ensures foundational concepts are mastered before advancing to more complex topics.

Identify prerequisite relationships between topics
Generate customized learning paths
Visualize topic dependencies through flowcharts

How to Use the PUP Curriculum Navigator

1. Select a Knowledge Area

Browse through the knowledge areas in the sidebar and click on any area you're interested in learning.

2. View Learning Path

The system will generate a personalized learning path for the selected knowledge area.

3. Follow the Sequence

Follow the recommended learning sequence to ensure you have all the necessary foundational knowledge.

4. Search for Knowledge Areas

Use the search bar to quickly find specific knowledge areas you're interested in.

This tool uses topological sorting to determine the optimal learning sequence based on topic dependencies.
